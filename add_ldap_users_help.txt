Adding Users to LDAP Server using ldapscripts command line utilities
--------------------------------------------------------------------

The following commands will need to be run as root or with sudo as the commands need to access the LDAP server password file.  You will need to run this commands with sudo.

1. Add a group:

  sudo ldapaddgroup testgroup


2. Then add a user to that group:

  sudo ldapadduser testuser1 testgroup


3. Then set that users password:

  sudo ldapsetpasswd testuser1

Enter password and password confirmation when prompted.
